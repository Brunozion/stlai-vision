# packages/shared

Pacote compartilhado entre frontend e backend.

## Objetivo

Centralizar contratos comuns para evitar divergencia entre camadas.

## Itens que devem morar aqui

- tipos de projeto
- enums de status
- schemas Zod
- DTOs compartilhados
- constantes de planos e creditos

## Estrutura sugerida

```text
src/
  contracts/
  schemas/
  types/
  constants/
```
